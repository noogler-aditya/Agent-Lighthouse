# Support

## Where to Get Help

- Usage and setup: open a GitHub Discussion or Issue
- Bug reports: use the bug report issue template
- Feature requests: use the feature request issue template
- Security issues: follow [SECURITY.md](SECURITY.md) (private report path)
- Operational incidents: use [docs/OPERATIONS.md](docs/OPERATIONS.md)

## What to Include

Please include:

- environment details (OS, versions)
- exact commands run
- expected vs actual behavior
- logs or screenshots

## Response Expectations

This is an open-source project. Response times vary based on maintainer availability.

## Triage Priority

1. Security and data exposure issues
2. Production outages and broken releases
3. Regressions in documented behavior
4. General support and enhancement requests
